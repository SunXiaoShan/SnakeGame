# SnakeGame
實作題目：實作貪食蛇 iOS App   
 實作項目:   
    * 有 Start 選項可以開始遊戲   
    * 死亡有 Try again 選項可以再次挑戰   
    * Gesture 控制上下左右移動   
    * 碰到身體或牆壁死亡    
    * 隨機產生食物位置   
    * 移動範圍為整個螢幕   
    * Model testing (XCTesting)
